linux/ubuntu上安装wordpress 步步为营

www.MyException.Cn   发布于：2012-08-24 10:00:20   浏览：1次
linux/ubuntu下安装wordpress 步步为营

安装apache2

1.打开一个终端 terminal

 

2.在终端上输入 以下内容 进行在线安装apache2

sudo apt-get install apache2

3.测试安装成功与否

打开浏览器 输入地址 http://localhost/

 

安装php5

4.在终端输入

sudo apt-get install php5 libapache2-mod-php5

5.为使PHP正常工作 重启apache

sudo /etc/init.d/apache2 restart

sudo /etc/init.d/apache2 restart

 6.测试apahce2 编写一个测试文件

命名为testphp.php

sudo gedit /var/www/testphp.php

 在文本编辑器中写入

<?php phpinfo(); ?>

 保存并关闭 然后去浏览器打开看效果 是否已经成功安装php环境

 

安装mysql5

7.输入以下内容

sudo apt-get install mysql-server

8.打开配置文件

gksudo gedit /etc/mysql/my.cnf

 修改bind-address = 127.0.0.1 为合适的ip

mysql -u root

mysql> SET PASSWORD FOR 'root'@'localhost' = PASSWORD('yourpassword');

安装phpmyadmin

 9.输入以下内容

sudo apt-get install libapache2-mod-auth-mysql php5-mysql phpmyadmin
 

10.为使php和mysql共同工作 修改php.ini文件

gksudo gedit /etc/php5/apache2/php.ini

 修改 ;extension=mysql.so 为 去掉前面的注释符 ；

11. 重启apache2

sudo /etc/init.d/apache2 restart

 

安装wordpres

在线下载最新版本wordpess压缩包

进行安装

1.拷贝解压的文件包到 /var/www下

cp －rf  源路径 /var/www

2.打开浏览器 进入phpmyadmin管理界面 新建一个数据库

例如命名为wordpress

3. 打开浏览器 本机测试为 http://localhost/wordpress

然后进行安装wordpress
