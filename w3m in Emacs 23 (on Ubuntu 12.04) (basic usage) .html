w3m in Emacs 23 (on Ubuntu 12.04) (basic usage)

Posted on December 18, 2012 by ChenShan

一.安装

Shell
1
sudo apt-get install w3m-el-snapshot

二.打开w3m(参考这里)

2.1.在emacs中打开

M-x w3m 打开w3m,进入默认主页

M-x w3m-find-file 用w3m打开本地文件

M-x w3m-browse-url  用w3m打开指定网页

2.2.在shell中打开

PHP
1
2
emacs -f w3m
emacs -f w3m http://emacs-w3m.namazu.org/

三.基本操作(参考这里)

3.1打开页面

<Enter>键 移动光标到链接处回车就进入了选定页面

g 在minibuffer打开一个url

G 在minibuffer打开一个url,但与g不同是打开了一个新会话(opens a new session)

c 显示当前页面的url

u 显示要打开的url

3.2刷新

R 重载页面(Reload the page)

三.页面内移动(参考这里)

C-n 或者 <SPC> 向下滚动

M-n 或者 <DEL> 向上滚动

> 滚动到左端

< 滚动到右端

.  向左滚动

,  向右滚动

M-l 使视野滚动到中间

<TAB> 移动到下一个超链接

S-<TAB> 移动到上一个超链接

]  移动到下一个表单

[  移动到上一个表单

}  移动到下一张图片

{  移动到上一张图片

四.页面间移动(参考这里)

B  后退一页(在历史页面中)

N  前进一页(在历史页面中)

H 回到主页面

^  回到当前页面的父目录,比如当前页面是"http://foo/bar/"然后要回到 "http://foo/"页面就可以用这个,特别在搜索的时候特别有用

<SPC>  当到达页面底端(at the end of the buffer),按空格键就会到达"next page",比如在谷歌搜索的时候底端会有类似 1 2 3 4 5 6 7 8... 可点击的链接,这时候只要按空格就会到达第2页,第3页,在搜索的时候特别有用

<DEL>  这个和<SPC>可以看成一组,也就是到前一页("previous" page)

五.使用鼠标

mouse-2  打开所指向的链接

S-mouse-2  在一个新的会话中打开所指向的链接

六.离开

q  关掉emacs-w3m窗口,选择另一个buffer

Q  保存已到达的URLs到磁盘,保存cookie,并真正退出emacs-w3m

七.缓冲区 (buffer)

7.1.创建与关闭

C-c C-t  用新名字创建一个当前活动缓冲区的副本,用于在一个新缓冲区里开启一个新会话,而不装载当前页

C-c C-w  关闭当前缓冲区

C-c M-w  关闭出当前活动缓冲区的其他缓冲区

7.2.移动

C-c C-p  移动到上一个缓冲区

C-c C-n  移动到下一个缓冲区

7.3.列表里选择缓冲区(参考这里)

C-c C-a  前进到minibuffer中的一个缓冲区(以名字为标识),用 M-p 与 M-n 在里表里移动,也可以用 <TAB>.(在输入名字的时候可以用<tab>键补全名字)

C-c C-s  在另一个窗口显示buffer列表,n 和 p 上下移动, <Enter>键确定.

八.书签(bookmarks) (参考这里)

a  添加当前页面到书签,如果有前缀名,则打开前缀名url并添加

M-a   添加当前指针(point)的url到书签

v  查看书签列表(The bookmark minor mode)

C-k  移除当前指针下的书签

E  访问当前书签文件

C-_   撤销最后一次操作

九.下载

d  现在当前指针所指的url资源

十.关于HTML表单(参考这里)

]  Jump to the next form (w3m-next-form).

[  Jump to the previous form (w3m-previous-form).

C-c C-c  Submit form at point (w3m-submit-form).

RET  Edit the value of the form item under point.

RET (w3m-form-*-keymap)Accept the value.

C-c C-q (w3m-form-*-keymap)Quit editing the form item, leaving changes.

十一.网页的编辑与hacking(参考这里)

\   Display the current web page in the raw HTML format(w3m-view-source).

=   Show the information about currently displayed web page. It includes title, URL, document type, last modified date(w3m-view-header).

E  Edit the local file pointed by URL of current page(w3m-edit-current-url).

e  Edit the local file pointed by URL under point(w3m-edit-this-url).

M  在外部浏览器中打开该URL
