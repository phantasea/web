ubuntu上安装wordpress博客系统网站的详细讲解

ubuntu下安装wordpress博客系统网站的详细讲解

wordpress已经是一个很受欢迎的php博客系统了，下面讲解一下在ubuntu环境下面怎么用wordpress搭建一个博客系统网站：

（1）、配置环境

       1、安装必要的软件

       打开终端：

       sudo apt-get install apache2

       sudo apt-get install libapache2-mod-php5 php5

       sudo apt-get install mysql-server-5.0  mysql-common mysql-admin

       sudo apt-get install php5-mysql

      2、mysql初始化配置

       通过命令sudo mysql-admin可进入mysql的图形界面，然后可以在里面设置一个数据库，作为后面建立网站的数据库，比如建立一个website的数据库作为后面搭建博客系统网站的数据库。

       也通过使用命令：mysql -u root -p回车，输入密码进入mysql的命令行界面，这样也可以使用命令行来建立相应的数据库。

      3、启动apache2和mysql  server

        sudo /etc/init.d/mysql start

        sudo /etc/init.d/apache2 start

 （2）、wordpress安装

        1、准备工作：

             下载wordpress，然后解压，放到apache2的目录里面（一般是在/var/www目录中），并修改wordpress文件夹的权限使用命令如下：

              wget -c http://wordpress.org/latest.tar.gz

              tar xvzf wordpress-2.2.tar.gz

              sudo cp -rf  wordpress /var/www

              sudo  chown -R www-data:www-data wordpress（修改权限）

    

         2、wordpress配置

              进入http://localhost/wordpress回车

             然后进行相应的数据库之类的设置即可完成wordpress的安装了。

当wordpress安装完成之后外网就可以通过相应的ip地址访问你自己的wordpress站点了。

后续：可以申请一个域名，让后把域名和ip地址给绑定，当ip地址和域名绑定之后就可以通过域名来访问自己的站点了。这样就可以通过在wordpress上面进行操作来实现搭建自己网站的目的了。
