List of Free Antivirus Software for Ubuntu 12.04 (Precise Pangolin)

Here is one question that has come up every time since Linux went mainstream: Do Linux users need antivirus software? There are two answers from two different camps. One group says, Linux OS doesn’t need antivirus as it is already secured, and not many threats are directed towards it. The other group will tell you that Linux is vulnerable like any OS, (Windows, Mac) and that adding another layer of security is a wise decision. I don’t know which of these groups you belong but I am going to show you how to install free antivirus software in Ubuntu. Programs like avast! for Linux, BidDefender, and AntiVir are all good programs to protect your systems against malicious software.

Objectives:

  • Install Antivirus in Ubuntu 12.04 (Precise Pangolin)
  • Enjoy!

 

Installing avast! Linux Home Edition in Ubuntu

avast! Linux Home Edition is a free antivirus software for your home computers, (Windows, Mac, Linux) It has a simple to use interface and automatically updates its virus definitions with newer ones. To install it in Ubuntu, press Ctrl – Alt – T on your keyboard to open Terminal. When it opens, run the commands below to download it.

http://files.avast.com/files/linux/avast4workstation_1.3.0-2_i386.deb

 

 

Next, run the commands below to install it.

sudo dpkg -i avast4workstation_1.3.0-2_i386.deb

 

 

When you launch it, it will prompt you to register and obtain a free license key.

 

Installing BitDefender in Ubuntu 12.04

BitDefender is another antivirus software that has a Linux version. To install it in Ubuntu, you first need to register for a free license key and the download link. Click here to register. After registering, open the Terminal, then copy and paste the line below to download BitDefender.

wget http://download.bitdefender.com/SMB/Workstation_Security_and_Management/BitDefender_Antivirus_Scanner_for_Unices/Unix/Current/EN_FR_BR_RO/Linux/BitDefender-Antivirus-Scanner-7.6-4.linux-gcc4x.i586.deb.run




 
 
Next, run the commands below to make the file executable.
sudo chmod +x BitDefender-Antivirus-Scanner-7.6-4.linux-gcc4x.i586.deb.run 

 
 
Finally, run the commands below to install it.
sudo ./BitDefender-Antivirus-Scanner-7.6-4.linux-gcc4x.i586.deb.run

 
 
Installing Avira AntiVir in Ubuntu 12.04
Avira AntiVir is another antivirus virus software that supports Linux, including Ubuntu. To install it in Ubuntu, run the commands below to download.
http://dlpe.antivir.com/package/wks_avira/unix/en/pers/antivir_workstation-pers.tar.gz

 
Then run the commands below to extract the downloaded file.
tar xvzf antivir_workstation-pers.tar.gz 

 
 
Finally, run the commands below to install it.
sudo ./antivir-workstation-pers-3.0.2-5/install 

 
 
Accept the agreement and select yes to all features.
 
This is my list. If you want to add more to it, comment on this post and add it.
Enjoy!
