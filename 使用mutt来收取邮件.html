使用mutt来收取邮件


配置mutt，@ubuntu

$ sudo apt-get install mutt fetchmail msmtp

$ vim ~/.muttrc
-----
set sendmail="/usr/bin/msmtp"
set use_from=yes
set realname="发信的名字"
set from=邮件地址
set envelope_from=yes
-----

$ vim ~/.fetchmailrc
----------
set postmaster "本机的UID"
set bouncemail
set no spambounce
set properties ""
poll pop3-ssl服务器地址 with proto POP3 port 995 uidl
user '邮箱用户' there with password '邮箱密码' is '本机的UID' here ssl sslproto
SSL23 mda "/usr/bin/procmail -d %T"
keep
----------

$ vim ~/.msmtprc
----------
account default
host smtp-ssl服务器地址
from 邮箱地址
tls on
tls_starttls off
auth on
port 465
timeout 15
user 邮箱用户
password 邮箱密码
logfile ~/.msmtp.log
tls_certcheck off
----------

之后执行如下命令来收取邮件：
$ /usr/bin/fetchmail
当然也可以放到你自己的crontab来自动执行！
