patool - Portable Archive File Manager - Linux CLI

patool - portable archive file manager

DESCRIPTION
Various archive formats can be created, extracted, tested, listed, searched, repacked and compared by patool. The advantage of patool is its simplicity in handling archive files without having to remember a myriad of programs and options.

EXAMPLES
patool extract archive.zip otherarchive.rar
patool --verbose test dist.tar.gz
patool list package.deb
patool --verbose create myfiles.zip file1.txt dir/
patool diff release1.0.tar.xz release2.0.zip
patool search "def urlopen" python-3.3.tar.gz
patool repack linux-2.6.33.tar.gz linux-2.6.33.tar.bz2

debtap - Convert .DEB files to Archlinux Packages - Linux CLI
Posted by: gotbletu
Follow
https://aur.archlinux.org/packages/debtap
https://github.com/helixarch/debtap

debtap - A script to convert .deb packages to Arch Linux packages, focused on accuracy. Do not use it to convert packages that already exist on official repositories or can be built from AUR!

http://pkgs.org/

# convert deb to archlinux pkg.tar.xz
debtap package_name.deb

# install package locally
sudo pacman -U package_name.pkg.tar.xz

# remove package
sudo pacman -R package_name

#list package
pacman -Qql package_name

#local package search
pacman -Qs package_name
