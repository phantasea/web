Vim Script List
列表和数组的概念
Vim Script 的列表是元素（这里的元素可以是数值、字符串、函数引用、列表）的有序集合，而数组则是存储列表的变量。在 Vim Script 里，这两个术语是常常混用的。不过更精确的来说，列表指的是数据，而数组指的是变量。列表的值不一定要放在数组里，但每个数组变量一定会包含一个列表（该列表可能是空的）。
1 列表中的元素是有序的，也就是说，从起始元素到终止元素的先后次序是固定的。
2 列表可以包含任意多个不同数据类型的元素。
3 列表中的每个元素都有相应的整数作为索引，从数字 0 开始递增，每次加 1。
列表的创建
列表使用方括号（[]）作为分隔符，里面的元素使用逗号（,）进行分隔。元素可以是数值、字符串、函数引用、列表。
列表忽略末尾额外的逗号。
空列表是直接使用方括号表示 []。
列表的语法形式：
let list = [element, element, ..., element]
例1：
let empty_list = []
let name_list = ['nana', 'nene', 'nini', 'nono', 'nunu']
let array_list = [1, '1', [1, 2, 3],]
列表的索引
列表中的每个元素都有相应的整数作为索引。我们正是通过这个索引来访问列表中的元素。
列表的第一个元素的索引为 0，向后则增加 1。列表的最后一个元素的索引为 -1，向前则减去 1。如倒数第二个元素的索引为 -2。
在列表之后的方括号中使用索引可以访问列表元素。如果元素本身是列表，则需要使用多重索引进行访问。
例1：
let family_member = [1, ['father', '28'], 2, ['mather', '28']]
echo 'index =' family_member[0]
echo 'age =' family_member[1][1]
echo 'index =' family_member[-2]
echo 'age =' family_member[-1][-1]
当使用非法的索引访问列表时，将出现错误。为了避免这种错误，可以使用 get() 函数。get() 函数在遇到非法索引时，返回指定的缺省值，如果没有缺省值，则返回 0。遇到合法的索引时，则返回索引所在的元素。
例2：
let scripts = ['perl', 'shell', 'python', 'ruby']
echo get(scripts, 3)
echo get(scripts, 5)
echo get(scripts, 5, 'none')
列表的访问
如果直接使用列表变量名称本身，则表示使用整个列表。
上一小节已经提到，需要通过索引来访问列表的元素。在列表变量之后的方括号中使用索引即可访问列表的元素。索引可以是整数，也可以是得到整数的表达式（表达式的结果不能是浮点数）。
列表访问的语法形式：
list[index_exp][index_exp]...[index_exp]
例1：
let numbers = [11, 10, 12, 9, 13, 8]
let i = 1
let j = 2
echo numbers[i + j]
echo numbers[i - j]
for 循环依次取列表中的元素赋值给指定的变量（可以是标量、也可以是列表）。
例2：
let numbers = [11, 10, 12, 9, 13, 8]
echo numbers
for element in numbers
echo element
endfor
for index in range(len(numbers))
echo numbers[index]
endfor
while 循环可以通过索引循环的方式来访问列表中的元素。
例3：
let numbers = [11, 10, 12, 9, 13, 8]
echo numbers
let index = 0
while index < len(numbers)
echo numbers[index] + 1
let index += 1
endwhile
map 函数是 for 循环的简化版，通过提供复杂的表达式来对列表进行计算（map 函数修改的是列表本身）。
例4：
let numbers = [11, 10, 12, 9, 13, 8]
echo numbers
call map(numbers, '(v:val > 10) ? v:val-2 : v:val+2')
echo numbers
子列表
Vim Script 的子列表是列表部分元素的集合。方法是在方括号内指定开始索引和结束索引，并使用冒号（:）分割首末索引。开始索引和结束索引可以是整数、表达式（结果必须是整数）、缺省。
子列表的语法形式：
let sub_list = list[s_index_exp : e_index_exp]
例1：
let numbers = [11, 10, 12, 9, 13, 8]
let sub_numbers = numbers[-2 : -1]
echo sub_numbers
let sub_numbers = numbers[2 : len(numbers)-1]
echo sub_numbers
开始索引如果省略等同于使用 0；结束索引省略等同于使用 -1。[ : ] 则表示取整个列表。
例2：
let numbers = [11, 10, 12, 9, 13, 8]
let sub_numbers = numbers[ : 4]
echo sub_numbers
let sub_numbers = numbers[2 : ]
echo sub_numbers
let sub_numbers = numbers[ : ]
echo sub_numbers
如果开始索引大于列表的最大长度或结束索引小于开始索引，则返回空列表。
例3：
let numbers = [11, 10, 12, 9, 13,8]
let sub_numbers = numbers[len(numbers) : 4]
echo sub_numbers
列表的赋值
就像标量值可以赋值给标量变量一样，列表也可以赋值给变量（通常是标量变量、列表变量）。
列表可以赋值给列表，也可以分别赋值给单独的变量，这相当于做了独立的赋值操作。
列表赋值的语法形式：
let list = o_list
let [var1, var2, ...; remain_var] = list
直接将一个列表赋值给另外一个列表，这两个列表实际上指的是同一列表（也可以这样说，在赋值的时候是使用列表的引用）。对其中一个列表的元素进行操作都会影响到另一个列表。但是如果对其中一个列表赋予新的列表值则不会影响到另一个列表。
例1：
let alist = [11, 10, 12, 9, 13, 8]
let blist = alist
echo alist blist
let blist[3] = 33
echo alist blist
let blist = [alist[0], alist[1]]
echo alist blist
可以使用操作符 is 检查两个变量是否指向同一个列表，如果相同返回真值。操作符 isnot 的功能刚好相反。操作符 == 比较两个列表的值是否相同。
两个列表使用操作符 is 得到的是真值，则两个列表使用操作符 == 得到的也必定是真值。
两个列表使用操作符 == 得到的是真值，则两个列表使用操作符 is 得到的不一定是真值。
例2：
let alist = [11, 10, 12, 9, 13, 8]
let blist = alist
let clist = [11, 10, 12, 9, 13, 8]
echo alist is blist
echo alist is clist
echo alist == clist
列表赋值给单个变量时，如果变量和列表的数量不一致，则会出错。要处理列表中所有额外的元素，可以加上分号（;）和一个列表变量。
例3：
let alist = [11, 10, 12, 9, 13, 8]
let [num1, num2; rest] = alist
echo num1
echo num2
echo rest
因为列表是在赋值运算开始之前建立的，所有在 Vim Script 里互换列表中的两个元素很容易。
例4：
let alist = [11, 10, 12, 9, 13, 8]
echo alist
let [alist[2], alist[3]] = [alist[3], alist[2]]
echo alist
列表的复制
Vim Script 支持把一个列表的值复制给另一个列表（不是指向同一个列表）。列表的复制可以分为浅复制和深复制。
浅复制和深复制的区别在于前者在复制列表的列表的时候，是复制的列表的引用，而后者采用递归的方式复制列表的值。
可以使用 [:] 和 copy 函数的方式进行浅复制。因此如果改变列表的列表的元素的值，则依然会改变另一个列表的值。
例1：
let alist = [[1, 2, 3], 1, 2, 3]
let blist = alist[ : ]
let clist = copy(alist)
let alist[0][0] = 11
let blist[0][1] = 12
let clist[0][2] = 13
echo alist is blist blist is clist
echo alist == blist blist == clist
echo alist[0] is blist[0] blist[0] is clist[0]
echo alist[0] == blist[0] blist[0] == clist[0]
可以这样理解，浅复制在复制列表的列表的时候实际上是复制列表的引用。因此改变列表的引用所指向的数据，两个列表都会有变化。但是如果将列表中的列表用另外一个列表赋值，则实际上赋值的是引用。这时两个引用指向不同的列表。
例2：
let alist = [[1, 2, 3], 1, 2, 3]
let blist = alist[ : ]
let blist[0] = [11, 22, 33]
echo alist blist
深复制是以递归的方式复制列表的值。可以使用 deepcopy 函数进行深复制。
例3：
let alist = [[1, 2, 3], 1, 2, 3]
let blist = deepcopy(alist)
let alist[0][1] = 11
echo alist blist
echo alist == blist
大部分操作列表的内置函数都会改变原来的列表，但我们的本意可能并不是这样，只是想得到操作后的结果。我们常使用 copy 函数或 deepcopy 函数创建原列表的拷贝，这样就不会改变原来的列表。
例4：
let alist = [[1, 2, 3], 1, 2, 3]
echo sort(deepcopy(alist))
echo alist
连接列表
两个列表可以使用列表连接操作符（+）连接成一个列表。
例1：
let alist = [[1, 2, 3]]
let blist = [1, 2, 3]
let clist = alist + blist
echo clist
let blist += [[4, 5]]
echo blist
修改列表
修改列表是对列表的某一个元素或部分元素进行修改（实际上是对列表的元素进行赋值）。
修改列表的语法形式：
let list[index_exp][index_exp]...[index_exp] = scalar or list
let list[s_index_exp : e_index_exp] = o_list
例1：
let alist = [[1, 2, 3], 1, 2, 3]
let alist[-1] += 10
let alist[0] = [11, 12, 13]r /> let alist[1 : 2] = [11, 12]
echo alist
增加和删除列表元素
增加和删除列表元素需要通过内置函数完成。元素可以是标量也可以是列表。
insert(list, element) 在列表的前面增加一个元素
insert(list, element, index_exp) 在列表中指定索引位置前增加一个元素
add(list, element) 在列表的最后增加一个元素
extend(list, o_list) 在列表的最后扩展一个列表
extend(list, o_list, index_exp) 在列表的指定索引位置扩展一个列表
remove(list, index_exp) 删除列表中指定索引位置的元素
remove(list, s_index_exp, e_index_exp) 删除列表中指定起始索引之间的所有元素
unlet list[index_exp] 删除列表中指定索引位置的元素
unlet list[s_index_exp : e_index_exp] 删除列表中指定起始索引之间的所有元素
例1：
let numbers = [11, 10, 12, 9, 13, 8]
call insert(numbers, 0)
call insert(numbers, [-1, -2])
call insert(numbers, 11, 3)
call add(numbers, [-11, -12])
call extend(numbers, [-21, -22])
echo numbers
call extend(numbers, [-23, -24], 3)
echo numbers
call remove(numbers, -1)
call remove(numbers, -2, -1)
unlet numbers[0 : 1]
unlet numbers[1]
echo numbers
其它操作列表的函数
Vim Script 的大部分列表操作都是通过内置函数来提供的，以实现排序、反转、范围、索引等操作。这些函数通常接受一个列表并返回列表的某些属性。
sort(list) 按 ASCII 码序给列表元素排序
reverse(list) 反转列表
len(list) 列表元素的个数，列表索引的最大值为 len(list) - 1
max(list) 列表元素的最大值
min(list) 列表元素的最小值
empty(list) 判断列表是否为空列表
count(list, element) 计算列表中元素出现的次数
index(list, element) 列表元素首次出现的索引
split(string) 按空白分离字符串保存到列表中
join(list, glue) 使用指定的内容连接列表的元素保存到字符串
string(list) 列表用字符串表示
map(list, exp) 对列表进行指定的表达式操作，列表中的元素依次赋值给 v:val
range(s_value, e_value, step) 自动产生数字列表，步进值可以为正负整数
range(e_value) 自动产生数字列表，默认从 0 开始，步进值为 1，最大值为 e_value - 1
例1：
let numbers = [11, 10, 12, 9, 13, 8]
echo 'sort=' sort(numbers)
echo 'reverse =' reverse(numbers)
echo 'len =' len(numbers)
echo 'max =' max(numbers)
echo 'min =' min(numbers)
echo 'empty =' empty(numbers)
echo 'count =' count(numbers, 2)
echo 'index =' index(numbers, 4)
echo 'join =' join(numbers, '<-->')
echo 'split =' split('hello world')
echo 'string =' string(numbers)
echo 'map =' map(numbers, 'v:val+3')
echo 'range =' range(4)
echo 'range =' range(8, 1, -2)
总结
列表篇出现的关键字（命令、操作符、函数名）有：
命令 ：let、unlet、call
操作符：is、isnot、==、+
函数名：get、copy、deepcopy、insert、add、extend、remove、sort、reverse、len
max、min、empty、count、index、split、string、join、map、range
