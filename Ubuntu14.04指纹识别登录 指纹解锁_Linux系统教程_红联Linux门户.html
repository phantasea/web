Ubuntu14.04指纹识别登录 指纹解锁

发布时间:2016-05-27 15:29:27来源:linux网站作者:luckylin666

指纹识别会让电脑避免输入密码的麻烦，下面我介绍一下Ubuntu系统实现指纹识别登录的几个步骤！电脑必须有自带指纹识别硬件。


1.首先更新一下系统更新包：

在终端中输入：sudo apt-get update


2.添加PPA源：

在终端中输入：sudo add-apt-repository ppa:fingerprint/fprint


3.安装指纹识别所需要的程序：

在终端中输入：sudo apt-get install libfprint0 fprint-demo libpam-fprintd gksu-polkit

上图所示已启用指纹登录。
