Additional/custom blocks which fit my needs

  • 源代码名称:i3blocks
  • 源代码网址:http://www.github.com/Anachron/i3blocks
  • i3blocks源代码文档
    • i3blocks源代码下载
  • Git URL: 
    git://www.github.com/Anachron/i3blocks.git
  • Git Clone代码到本地: 
    git clone http://www.github.com/Anachron/i3blocks
  • Subversion代码到本地: 
    $ svn co --depth empty http://www.github.com/Anachron/i3blocks
Checked out revision 1.
$ cd repo
$ svn up trunk

    • i3blocks

    这已经成为i3wm所使用的巨大的块集合。 它也可以是在bash中如何做事情的灵感。

    要求

    所有脚本都是用bash编写的。 每个脚本都应该针对性能进行优化。

    警告：

    某些脚本( 如显示) 也会更改计算机的行为 ！ 请将这些内容包含到i3blocks中时请非常小心。
