w3m-img for FreeBSD framebuffer

w3m-img for FreeBSD framebuffer とは

w3m-img 0.5.2 を FreeBSD のフレームバッファに対応させるパッチです。

w3m-img for FreeBSD framebuffer を使用するには

FreeBSD 6.3-STABLE で、w3m-img for FreeBSD framebuffer を使用するまでの設定を解説します。

w3m-img for FreeBSD framebuffer をインストールする

手動でビルドすることに抵抗がある人もいると思いますので、Ports Collection に対応した形式になっています。

次の手順にしたがって w3m-img for FreeBSD framebuffer をインストールしてください。 

% fetch http://www.ac.auone-net.jp/~baba/w3m-img/w3m-img-0.5.2-FreeBSD-0.1.3.tar.gz
% tar xvzf w3m-img-0.5.2-FreeBSD-0.1.3.tar.gz
% cd w3m-img-0.5.2-FreeBSD-0.1.3
% su
# cp Makefile.local /usr/ports/www/w3m
# cp files/patch-* /usr/ports/www/w3m/files

以上の手順が完了してから、Ports Collection の japanese/w3m-img、www/w3m-m17n-img、または、www/w3m-img のいずれかをインストールして、jfbterm for FreeBSD 上で実行してみてください。
